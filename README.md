## 实现预想

```
   功能路由：Twilight+Dispatcher
   功能模块化：Saya,使用例参考ABot-Graia
   聊天功能：独立于功能路由之外，也可以使用Twilight
   帮助文档：可配合模块管理写入配置文件
   开机启动：区分systemd与openrc
```
